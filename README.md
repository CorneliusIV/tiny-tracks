# Tiny Tracks
A simple API to return a song from Tiny Desk artists


## Requirements
- Node >12.0.0

## Local Setup

###  Installation
1. Clone repo
2. `nvm use`
3. `npm install`
4. `cp example.env .env`


### Set up environment variables
5. `cp example.env .env`

### start services
6. `node server.js`

Visit (http://localhost:3000)
